# 第三方库的pip安装方法
    - pip -h
    - 适合99%以上的安装需求，但需要网络

# 第三方库的集成安装方法
    - 批量安装库
    - 数据处理方面的集成开发工具：Anaconda集成开发工具
      - https://www.anaconda.com/

# 第三方库的文件安装方法
    - 某些第三方库需要先编译再安装
    - 不会编译？
      - 找网上已经编译好的文件
      - 比如UCI页面（https://www.lfd.uci.edu/~gohlke/pythonlibs/）
    - 下载好后，使用 pip install <文件名> 安装
